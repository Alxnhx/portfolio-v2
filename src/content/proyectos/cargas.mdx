## Contexto

<div className="context-grid">
  {/* Columna izquierda: texto + terminal */}
  <div className="context-main">
    Un sistema interno para gestionar cargas de entrada y salida que originalmente se registraban en hojas sueltas, correos y formatos impresos. Cada carga podía tardar cerca de 5 minutos en capturarse entre papeles, firmas y validaciones manuales, lo que generaba errores, pérdidas de información y tiempos muertos al intentar reconstruir el historial.

El objetivo: que el personal operativo y administrativo pueda ver en un solo panel qué cargas están entrando, cuáles están saliendo, en qué estado se encuentran y con qué datos fueron registradas, dejando la captura en alrededor de minuto y medio por carga gracias a un lector de huelas biometrico y asi reduciendo al mínimo la pérdida de información.

<script type="module">{`
  const initCarousel = () => {
    const carousel = document.querySelector('.project-carousel');
    if (!carousel) return;

    const slides = carousel.querySelectorAll('.project-carousel-slide');
    const thumbs = carousel.querySelectorAll('.project-carousel-thumb');
    const prevBtn = carousel.querySelector('.prev');
    const nextBtn = carousel.querySelector('.next');
    
    let currentIndex = 0;

    const showSlide = (index) => {
      currentIndex = ((index % slides.length) + slides.length) % slides.length;
      
      slides.forEach((slide, i) => {
        slide.classList.toggle('is-active', i === currentIndex);
      });
      
      thumbs.forEach((thumb, i) => {
        thumb.classList.toggle('is-active', i === currentIndex);
      });
    };

    prevBtn?.addEventListener('click', () => showSlide(currentIndex - 1));
    nextBtn?.addEventListener('click', () => showSlide(currentIndex + 1));

    thumbs.forEach((thumb, index) => {
      thumb.addEventListener('click', () => showSlide(index));
    });
  };

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initCarousel);
  } else {
    initCarousel();
  }
`}</script>
